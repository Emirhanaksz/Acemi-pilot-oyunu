# 🛩 Acemi-pilot-oyunu
# 🔎 HAKKINDA
Bu proje; HTML, CSS ve JavaScript teknolojileri kullanılarak Web Tabanlı Programlama dersi ödevi için hazırlanmış bir 2D oyundur.
